Run the program by calling {\ttfamily ./minesweeper 10x10 20} ~\newline
 The first argument must be a 3 or 5 character matrix, e.\+g. {\ttfamily 5x5} or {\ttfamily 15x15}. ~\newline
 The second argument must be a one or two digit number of mines for the game, e.\+g. {\ttfamily 5} or {\ttfamily 15}. ~\newline
 \hypertarget{index_autotoc_md15}{}\doxysubsection{Runtime}\label{index_autotoc_md15}
First, the user has to enter a name. If the player is known, statistics will be loaded. ~\newline
 Afterwards, the matrix will be generated and the user can start to play. ~\newline
 The user can choose fields in the matrix by writing the column name and the row number. He can also add a question mark in front of it, to mark that field as a mine. ~\newline
 Examples\+: {\ttfamily A2, D10, ?B4, ?C15} ~\newline
 \hypertarget{index_autotoc_md16}{}\doxysubsection{How to win}\label{index_autotoc_md16}
You can win by either opening all correct fields or by marking all mines correctly. ~\newline
